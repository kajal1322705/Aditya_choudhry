# Aditya Choudhry - Portfolio

## Overview
An interactive, professionally animated portfolio website showcasing projects and skills. Built entirely with vanilla HTML, CSS, and JavaScript with smooth animations, modern design, and vibrant colorful effects.

## Project Structure
```
.
├── index.html              # Main HTML structure with semantic sections
├── blog-post.html          # Blog post template
├── styles.css              # Complete styling with animations and responsive design
├── script.js               # Legacy JavaScript (fallback)
├── server.py               # Simple Python HTTP server on port 5000
└── js/
    ├── app.js              # Main application entry point (ES6 modules)
    └── modules/
        ├── analytics.js        # Page view and scroll tracking
        ├── back-to-top.js      # Back to top button
        ├── cartoon-assistant.js # Animated assistant
        ├── command-palette.js   # Command palette (Ctrl+K)
        ├── contact-form.js      # Contact form with validation
        ├── github-widget.js     # GitHub activity widget
        ├── keyboard-shortcuts.js # Keyboard shortcut handler
        ├── language-switcher.js  # Multi-language support
        ├── lazy-loading.js      # Image lazy loading
        ├── load-animations.js   # Load animations
        ├── navigation.js        # Navigation handling
        ├── particle-animation.js # Particle background
        ├── preloader.js         # Page preloader
        ├── projects.js          # Project filtering
        ├── reading-progress.js  # Reading progress bar
        ├── scroll-animations.js # Scroll-based animations
        ├── search.js            # Global search
        ├── skills.js            # Skills section
        ├── smooth-scroll.js     # Smooth scrolling
        ├── theme-toggle.js      # Dark/light mode
        └── typing-effect.js     # Typing animation
```

## Features
1. **Animated Hero Section**: Particle background, typing effect, rainbow gradient text
2. **Interactive Projects Grid**: 12 projects with filtering by technology
3. **Smooth Scroll Animations**: Fade-in and slide-in effects on scroll
4. **Responsive Design**: Mobile-optimized with touch-friendly interactions
5. **Custom Animations**: Floating orbs, gradient shifts, card hover effects
6. **Command Palette**: Quick navigation with Ctrl+K
7. **Multi-language Support**: English and Hindi
8. **SEO Optimized**: Meta tags, Open Graph, structured data
9. **Analytics**: Page views and scroll depth tracking
10. **Contact Form**: With validation and email client integration
11. **GitHub Widget**: Activity visualization
12. **Certifications Gallery**: With verification links

## Keyboard Shortcuts
- `Ctrl/Cmd + K` - Open command palette / search
- `?` - Show keyboard shortcuts help
- `Esc` - Close overlays
- `↑↓` - Navigate search results

## Development
The site is served using Python's built-in HTTP server on port 5000.
All animations and interactions run client-side with no backend dependencies.
