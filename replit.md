# Aditya Choudhry - Portfolio

## Overview
A clean, basic portfolio website built with vanilla HTML, CSS, and JavaScript. Simple design focused on content and usability.

## Project Structure
```
.
├── index.html          # Main HTML page
├── styles.css          # Styling
├── app.js              # Simple JavaScript (mobile menu, smooth scroll)
├── server.py           # Python HTTP server on port 5000
└── replit.md           # Documentation
```

## Sections
1. **Hero**: Introduction with name, title, and social links
2. **About**: Brief bio with stats
3. **Skills**: 8 technology cards
4. **Projects**: 6 project cards
5. **Contact**: Email and social links
6. **Footer**: Copyright

## Technologies
- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
