# Aditya Choudhry - Portfolio

## Overview
A modern, interactive portfolio website with multiple color themes, smooth animations, particle effects, and contemporary design. Built with vanilla HTML, CSS, and JavaScript featuring a unified architecture.

## Recent Changes
- **December 1, 2025**: Complete Modern Restructure
  - Rebuilt entire portfolio with new modern architecture
  - Implemented 5-theme system: Midnight, Ocean, Forest, Sunset, Neon
  - Added animated preloader with "AC" logo and gradient rings
  - Created smooth navigation with mobile overlay menu
  - Built particle background that adapts to current theme colors
  - Added floating gradient shapes and grid pattern background
  - Implemented typing animation effect in hero section
  - Created theme switcher panel with visual theme previews
  - Added search modal with smooth animations
  - Built project detail modals with glassmorphism design
  - Consolidated all JavaScript into unified app.js (replaced ES6 modules)
  - Enhanced responsive design for all screen sizes

## Project Structure
```
.
├── index.html          # Main HTML structure (semantic sections, modals, theme panel)
├── styles.css          # Complete styling (themes, animations, responsive design)
├── app.js              # Unified JavaScript (all functionality in one file)
├── server.py           # Python HTTP server on port 5000
└── replit.md           # Project documentation
```

## Features
1. **5 Color Themes**: Midnight (purple), Ocean (cyan), Forest (green), Sunset (orange), Neon (pink)
2. **Animated Preloader**: Gradient "AC" logo with spinning rings
3. **Hero Section**: Typing effect, gradient text, "Available for work" badge
4. **Particle Background**: Dynamic particles that match current theme
5. **Floating Shapes**: Animated gradient circles and orbs
6. **Smooth Navigation**: Gradient buttons, mobile overlay, scroll effects
7. **Theme Switcher**: Side panel with visual theme previews
8. **Search Modal**: Global search with keyboard shortcut (Ctrl+K)
9. **Project Modals**: Glassmorphism design with animations
10. **Skills Section**: Filterable skill cards by category
11. **Experience Timeline**: Visual timeline with animations
12. **Contact Form**: With email client integration
13. **Scroll Animations**: Fade-in effects on scroll

## Theme System
Each theme customizes:
- Primary and secondary colors
- Gradient combinations
- Background colors
- Text colors
- Accent colors

Themes are stored in localStorage for persistence.

## Technologies Used
- HTML5 (Semantic structure)
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (Classes, Event handling)
- Canvas API (Particle system)
- Intersection Observer API (Scroll animations)

## Keyboard Shortcuts
- `Ctrl/Cmd + K` - Open search modal
- `Esc` - Close modals/overlays

## Development
The site is served using Python's built-in HTTP server on port 5000.
All animations and interactions run client-side with no backend dependencies.
