## Project Structure
- `index.html` - Main HTML structure with semantic sections
- `styles.css` - Complete styling with animations and responsive design
- `script.js` - Interactive JavaScript features
- `server.py` - Simple Python HTTP server on port 5000

## Features
1. **Animated Hero Section**: Particle background, typing effect, gradient text
2. **Interactive Projects Grid**: 12 projects with filtering by technology
3. **Smooth Scroll Animations**: Fade-in and slide-in effects on scroll
4. **Responsive Design**: Mobile-optimized with touch-friendly interactions
5. **Custom Animations**: Gradient shifts, card hover effects, parallax scrolling

## Technologies Used
- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- Intersection Observer API
- Canvas API for particle system
